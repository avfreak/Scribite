Reviews
  • Reviews: PostNuke ermöglicht es "Reviews" (Rezensionen) auf der Seite zu erstellen. Mit diesem Modul werden die Reviews verwaltet. Zunächst gibt man einen Titel ein, z.B. "Wir brauchen Ihre Reviews". Als nächstes geben sie eine "Beschreibung" der Reviews ein, z.B.: "Dieser Bereich enthält Reviews folgender Themen: Gaststätten, DVDs, CDs, Bücher, Artikel, Kleidung, Veranstaltungen in unserer Stadt, Sportveranstaltungen, ..... haben Sie also letztlich in einem schrecklichen Restaurant gegessen oder waren bei einer tollen Veranstaltung, teilen Sie uns einfach Ihre Meinung mit, indem Sie einen Rezension schreiben."
    Es kann zu einem Review ein Bild angezeigt werden, welches aber nicht größer als 150x150 Pixel sein sollte. Das Bild muß in modules/Reviews/images gespeichert werden. Gibt es bereits Reviews, die auf eine Freischaltung warten, so erscheinen diese unten auf der Seite. Der Administrator kann jeden Review "editieren, ändern, das Datum stellen" wenn er auf "Speichern" klickt. Drückt er auf "Löschen", wird der Review (WICHTIG) ohne Bestätigung gelöscht.
    Um selber einen Review zu schreiben, klickt man auf "hier klicken um einen Review zu verfassen". Hier kommt man dann auf eine Seite, in der alle Reviews alphabetisch und in einer Toplist angezeigt werden. Mit Review eintragen gelangt man auf die Seite in der man einen Titel, eine Beschreibung, eine URL und eine Grafik angeben kann, die diesem Review zugeordnet werden. Nachdem man sich die Vorschau angesehen hat, kann man als Admin den Review direkt in die Liste eintragen lassen. Bitte auf korrekte Schreibweise und auf richtige Grammatik achten. "Abbrechen" klicken, wenn die Bearbeitung abgebrochen werden soll.
